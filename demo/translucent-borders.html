The Problem

You want to show a semi-transparent border around a text area that is on top of a background image. The text area has a green background. The semi-transparent border should be a translucent layer on top of the background image.

The Solution

Instead of using this:

  • border: 10px solid hsla(0,0%,100%,0.5);
  • max-width: 25vw;
  • background-color: white;

This should have had semi-transparent borders without green hue.

You can use this:

  • border: 10px solid hsla(0,0%,100%,0.5);
  • max-width: 25vw;
  • background-color: white;
  • background-clip: padding-box; /* Newly added */

This should have semi-transparent borders without green hue. Do you see it now?
