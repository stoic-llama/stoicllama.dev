The Problem

You want to use a picture as a background for borders with the border-image property.

The Solution

There are a few attributes in the border-image property. If you look at the official CSS Spec for border-image, you will find several key attributes for this property. Example below.

  • border: 20px solid black;
  • border-image-source: url(/img/circles.png);
  • border-image-slice: 20;
  • border-image-width: 20px;
  • border-image-repeat: round;

border-image-slice property refers to the "inward offsets" from the top/right/bottom/left edges of the image.

border-image-width property refers to the "inward distances" from the top/right/bottom/left sides of the border image area. Note If two opposite border-image-width offsets are large enough that they overlap, then the used values of all border-image-width offsets are proportionally reduced until they no longer overlap.

border-image-repeat property refers to how the images for the sides and the middle part of the border image are scaled and tiled (i.e., stretch, repeat, round, space).

Effectively slice draws the 9 regions that parts of the image will be used. Width is used to draw the parts of the image within the sliced regions. Repeat is also used to draw the parts of the image within the sliced regions.

Final notes:
  • The sample circles.png file is 60*60 px dimensions, where each circle is 20 out of 60 (3 circles in rows/cols). So to get one circle painted into each sliced region, you need to define slice by 20px for painting surface, and width by 20px of the content to be displayed.
  • Make sure to add a border property to define the area where you can use border-image to play with. If you do not define border the border-image will not display.
